Android Studio 設定 2.2.3版

  • AndroidStudio

Android Studio安裝後的設定

右下方的 Configure --> SDK Manager -->下方的 Launch Standalone SDK Manager

勾選完，按右下角的Install

更改快捷鍵組 -- 改成Eclipse的快捷鍵

右下方的 Configure --> Settings

將語系改成 UTF-8

當滑鼠移動到方法名稱上的時候會有提示文字

其它

顯示方法分隔符
顯示空格

optimize imports on the fly
add unambiguous imports on the fly

黑色背景主題：Darcula

  1. File --> Settings --> Editor --> Colors & Fonts -->
  2. 設定成:: Scheme name: Darcula

自訂字型、大小、行距

  1. File->Settings->Editor->Colors & Fonts-> 選好主題
  2. Save As... 另存主題 (要取名)
  3. 左欄位Colors & Fonts 點開
  4. 設定：(可自行變換)
    Primary font: Source Code Pro
    Size: 16
    Line spacing: 1.1

顯示行號

  1. File --> Settings
  2. 左欄位 Editor --> General --> Appearance，右欄位，勾選 Show line numberss
