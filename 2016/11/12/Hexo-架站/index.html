切到 blog 資料夾

開伺服器，看網頁狀況

安裝hexo托到github的工具

修改 _config.yml 檔案 在deploy:下加一些東西

使用Github SSH Key， 免去Hexo部署时輸入密碼，列出已存在的SSH Key。

生成SSH

執行後會生成id_rsa.pub文件，畫面會提示檔名，開啓檔案，裡面就是SSH key的内容，key的內容會是以ssh開頭的文字，把內容copy起來，要貼到github。
進入github ==> Seetings ==> SSH and GPG keys ==> New SSH keys ==> Title自取 ==> 把key內容貼到 Key 裡 ==> Add SSH keys ==>完成

判斷SSH是否配置好

出現以下文字就表示完成了 Hi huang2189! You've successfully authenticated, but GitHub does not provide shell access.

執行一次

第一次使用會詢問github的帳號跟密碼 可以到http://github帳號.github.io看網頁生成的狀況
