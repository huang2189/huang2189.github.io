Android 專案更名 for Android Studio

Android
  • AndroidStudio

更改專案名稱，有四個地方要改。

  • 專案資料夾
  • 套件名稱
  • Gradle的應用程式ID
  • string.xml

專案資料夾

  • 直接修改專案資料夾名稱。
  • 還可以直接到專案資料夾裡的.idea.name檔案內修改檔名。(沒試過，2.2.3版的沒看到)

套件名稱

更改套件名稱會影响所有的套件。

直接在package 上點右鍵選 Refator -> Rename

會出現警告對話框

按 Rename Package 後，更改名稱

會快速跑一下，但名稱還不會改，請選下面的 Do Refactor

Gradle的應用程式ID

File -> Project Structure

value\string.xml

雙擊string.xml，修改app_name的字串
