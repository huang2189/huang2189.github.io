Oracle資料查詢--設定查詢條件與資料排序

資料庫
  • oracle

SQL命令的分類

資料查詢

過濾資料列--WHERE子句

未過濾

過濾--deptno只留10的

WHERE子句說明

  • 只有當條件子句結果為真時，才會將該筆資料傳回。
  • 條件子句：
    • 邏輯條件
    • 比較運算式
    • 邏輯運算式
    • SQL特定運算式

邏輯條件

真(True)/假(False)/NULL

比較運算式

字串資料--找出JAMES的資料

數值資料 -- 找出薪資大於或等於2000

日期時間資料 -- 1981以後加入公司的

邏輯運算子

AND必須二個運算元都是True才會符合查詢條件

薪資大於等於1100 AND 職稱為clerk

OR -- 只要有一個運算元是True就符合查詢條件

薪資大於等於1100 或 職稱為clerk

NOT -- 反向運算

取出職稱非CLERK的

SQL特定運算子

  • 範圍區間的搜尋，使用BETWEEN運算子。
  • 列舉可能的資料串列，使用IN運算子。
  • 字串樣版搜尋資料，使用LIKE運算子。(只能用在字串)
  • 使用IS NULL運算子來檢測NULL值。

BETWEEN AND 運算子 -- 連續區間的條件判斷

IN 運算子 -- 列舉值的查詢

直接指定條件

LIKE 運算子 -- 模糊比對 -- 萬用字元查詢(模糊比對)

  • % 任意字元
  • _　 一個字元

IS NULL 運算子

  • 判斷資料是否為 NULL 值
  • IS NULL用於判斷資料項是否為 NULL 值的專屬運算子

資料排序 -- ORDER BY 子句

  • ASC 升幂，由小到大排序(Ascending)
  • DESC 降幂，由大到小排序(Descending)

可排序資料

欄位 / 別名 / 運算式 / 位置。

預設資料排序方式 -- 預設由小到大

  • 數值資料：由小到大。
  • 日期資料：依世紀、年、月，日，時，分，秒，由小排到大。
  • 字元資料：依內碼順序由小到大排序。
  • 空值：排序時，將NULL值視為最大值。

範例 -- 升冪(由小到大)預設

範例 -- 降冪(由大到小)

範例 -- NULL為最大值

使用欄位位置排序

  • 依資料項在SELECT 中的位置順序

多個資料項排序

  • 每一個資料項(欄位)皆可指定降冪或升膜冪
