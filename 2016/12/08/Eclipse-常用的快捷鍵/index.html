Eclipse 常用的快捷鍵

Eclipse
  • Eclipse

輔助程式碼： Alt + /
程式碼格式化排列整齊： Ctrl + Shift + F
自動import 1/2 ： Ctrl + Shift + M [Add import]
自動 import 2/2 ： Ctrl + Shift + O [Organize imports]
刪除一行： Ctrl + D
搜尋 1/2： Ctrl + K
搜尋 2/2： Ctrl + F
註解單行或多行(Java code)： Ctrl + 7
註解單行或多行(JSP)： Ctrl + Shift + C
往下移動： Alt + ↓
往上移動： Alt + ↑
往下複製： Ctrl + Alt + ↓
往上複製： Ctrl + Alt + ↑
同步變數更名： Alt + Shift + R
折疊所有代碼： Ctrl + / (小鍵盤)
展開所有代碼： Ctrl + * (小鍵盤)
快速修正： Ctrl + 1 (天下第一鍵)
所有快捷鍵： Ctrl + Shift + L
